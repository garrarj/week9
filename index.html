Video

Common video file formats

  • .MP4

    The .MP4 video file format is one of the most common and is compatible with an enourmous range of players. Many video streaming services, like Youtube, Vimeo, etc., prefer the .MP4 format

  • .MOV

    MOV was developed by Apple as a file format to use with their Quicktime player. MOV files are usually high quality and large in size

  • .AVI

    One of the oldest and must universally compatible video file formats. AVI tends to be large files.

  • .WMV

    WMV (Windows Media Videos) can be smaller in size, but are often poorer quality compared to the other formats above.

Audio

Common audio file formats

  • .WAV

    WAV stands for 'Waveform Audio File Format' and tend to be large files due to their uncompressed format

  • .AIFF

    AIFF stands for 'Audio Interchange File Format' and was developed by Apple in 1988. They tend to be large files due to their uncompressed format

  • .MP3

    MP3 stands for 'MPEG-1 Audio Layer 3' and is one of the most common audio file formats. MP3 files have nearly univeral compaitibility

  • .FLAC

    FLAC stands for 'Free Lossless Audio Codec. It is a lossless compression file format, so it can feature smaller file sizes than uncompressed formats while maintaining a high degree of audio fidelity.
